Básico HTML

Em HTML podemos escrever títulos com "<hx>titulo<hx/>", com x podendo variar de 1 a 6. Como podemos ver o título "HTML" acima.

Também podemos colocar imagens com "<img src= (link da imagem) height="altura da imagem, mas escolhendo a altura a largura ja fica proporcional" alt="texto alternativo da imagem"/>. Imagem de exemplo abaixo.

Também há como fazer listas, com <ul>lista desordenada</ul> ou <ol>lista ordenada</ol>, sendo os elementos das listas elementos <li>elemento 1</li>. Exemplo abaixo.

    lista:
  • elemento 1
  • elemento 2
  • elemento 3

    • Links são colocados como <a href="link">Nome do link</a>
